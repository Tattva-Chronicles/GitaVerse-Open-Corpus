# Contributing to GitaVerse-Open-Corpus

Thank you for your interest in contributing!
This project aims to create the largest **open, structured, machine-learning-ready** corpus of the Bhagavad Gita — including verses, translations, commentaries, and modern Q&A datasets.

## Ways You Can Contribute

### 1. Improve Data Quality

- Fix typos or transliteration errors
- Improve metadata accuracy
- Add verse mapping validations
- Add missing languages

### 2. Enhance Scripts

- Improve preprocessing efficiency
- Add new data loaders
- Improve semantic clustering
- Add vector search utilities
- Add Jupyter notebooks for demos

### 3. Add New Derived Datasets

- Sanskrit → Transliteration models
- Hindi/English paraphrase datasets
- Question generation datasets
- RAG-friendly verse embeddings

### 4. Documentation

- Add tutorials
- Add schema diagrams
- Improve READMEs

---

## Contribution Flow

1. Fork the repository
2. Create a feature branch
3. Commit changes with meaningful messages
4. Submit a Pull Request

---

## Coding Guidelines

- Use Python 3.10+
- Follow PEP-8
- Use type hints wherever practical
- Ensure scripts can run standalone
- Keep dataset processing functions pure and deterministic

---

## Dataset Guidelines

- Do **not** upload any copyrighted data
- Only include **processed**, transformed, or canonical versions
- Always include attribution for CC-BY datasets
- Follow the structure defined in `DATASET_ARCHITECTURE.md`
